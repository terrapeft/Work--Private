Installation notes:

1) Open ddl script for the Users db.
	1.1) Check/edit the path where the db files will be created.
	1.2) Update app pool name, currently: IIS APPPOOL\TradeDataAppPool
	1.3) Run the script

2) Run jobs' scripts.

3) Run data script.



Configuring via UI

1) Group API methods in the Service Configuration/Method Groups.
2) Group search columns in the Service Configuration/Search Groups.
