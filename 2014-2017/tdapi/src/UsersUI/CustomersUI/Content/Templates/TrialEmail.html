How the URL is structured:
  • tda stands for the database alias, during the trial period, this is the only DB you can use.
  • GetExchangeCodes - method name
  • csv - data format
  • then go parameters and filters

There are a number of available commands and options:
  1. List all available methods with parameters:
    {ServiceUrl}tda/list?u={username}&p={password}
  2. When you have a list of methods, you can query for the method help, for example lets query method GetExchangeCodes:
    {ServiceUrl}tda/{methodName}/help?u={username}&p={password}
  3. And you can count number of results:
    {ServiceUrl}tda/{methodName}/count?u={username}&p={password}
  4. Query a method for data in:
    • csv format:
      {ServiceUrl}tda/{methodName}/csv?u={username}&p={password}
      • add column headers:
        {ServiceUrl}tda/{methodName}/csv?cn=true&u={username}&p={password}
      • force quotes:
        {ServiceUrl}tda/{methodName}/csv?cn=true&q=true&u={username}&p={password}
      • specify custom delimiter:
        {ServiceUrl}tda/{methodName}/csv?cn=true&q=true&d=;&u={username}&p={password}
    • xml format:
      {ServiceUrl}tda/{methodName}/xml?u={username}&p={password}
    • json format:
      {ServiceUrl}tda/{methodName}/json?u={username}&p={password}
      • add indents:
        {ServiceUrl}tda/{methodName}/json?ij=true&u={username}&p={password}
  5. Specify page size and page number:

    pg - page size,
    pn - page number

    Show first 50 records:

    {ServiceUrl}tda/{methodName}/help?u={username}&p={password}&ps=50&pn=1
  6. Specify filters.

    Available filters are:
    Equal (=):
    t0.ColumnName=Value
    		 Greater than (gt): 
    t0.ColumnName-gt=Value
    Less than (lt): 
    t0.ColumnName-lt=Value
    		 Greater than or equal (gte): 
    t0.ColumnName-gte=Value
    Less than or equal (lte): 
    t0.ColumnName-lte=Value
    		 LIKE: SQL "like" operator, use "%" as a wildcard, you MUST use URL encoded percent sign - "%25", 
    t0.ColumnName-like=%25Value%25, t0.ColumnName=Value%25
    IN: SQL "in", separate values with comma, without quotes, 
    t0.ColumnName-in=new year,old school,red fox
    		 NOTIN: SQL "not in" same rules like for "in", 
    t0.ColumnName-notin=new year,old school,red fox
    SORT: you can order output by column or by several columns, 
    t0.ColumnName-sort=ASC&t0.AnotherColumnName-sort=DESC
    		 AND: concatenate current condition with previous one, "and" is a default value and can be omitted, 
    t0.ColumnName=14&t0.AnotherColumnName=AMEX
    OR: 
    t0.ColumnName=14&t0.or-AnotherColumnName=AMEX
  7. You can specify several filters for one column.



You can call the service from your application.
C# example : 
using System;
using System.Net;
using System.Net.Http;
using System.Net.Http.Headers;
namespace TdApiExample
{
    class Program
    {
        static void Main(string[] args)
        {
            DownloadData();
        }
        static private void DownloadData()
        {
            using (var client = new HttpClient())
            {   
                // this is a base URL
                client.BaseAddress = new Uri("{ServiceUrl}");
                client.DefaultRequestHeaders.Accept.Clear();
                client.DefaultRequestHeaders.Accept.Add(new MediaTypeWithQualityHeaderValue("application/json"));
                client.DefaultRequestHeaders.Accept.Add(new MediaTypeWithQualityHeaderValue("text/xml"));
                client.DefaultRequestHeaders.Accept.Add(new MediaTypeWithQualityHeaderValue("text/plain"));
                HttpResponseMessage response = null;
                try
                {
                    response = client.GetAsync("tda/{methodName}/csv?cn=true&u={username}&p={password}").Result;
                    response.EnsureSuccessStatusCode(); // treat http error codes as exceptions
                    var content = response.Content.ReadAsStringAsync().Result;
                    Console.WriteLine(content);
                }
                catch (HttpRequestException rex)
                {
                    var status = response?.StatusCode;
                    if (status != null)
                    {
                        Console.WriteLine($"{(int) status} {status}");
                    }
                    else
                    {
                        Console.WriteLine(rex);
                    }
                }
                catch (Exception ex)
                {
                    Console.WriteLine(ex);
                }
            }
        }
    }
}
